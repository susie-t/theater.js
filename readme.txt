*theater.jsとは

-theater.jsはWEBページ上でアドベンチャーゲームを作成するためのJavaScriptフレームワークです。

-主な機能

--画面クリック等のイベントにあわせ、シナリオに沿って画像と文章を連動表示します。

--文章をタイプライトしたように表示します。

--画像の表示時に、効果(フェードアウト・イン等)を設定できます。

--メニューを表示し、プレイヤーの入力を取得します。

--条件により、シナリオを分岐させることができます。

-ブラウザのみで動作します(推奨:WinXp+IE6)。Flash等のプラグインは必要ありません。

-prototype.js(v1.5.0)、およびscript.aculo.us(v1.7.0)のdragdrop.js、effects.jsを利用しています。また、prototype.jsについては独自に機能を拡張しています。

-公開当初はアドベンチャーゲーム風の表示ができるのみでしたが、v2.0.0からメニュー表示機能やシナリオ分岐機能を備え、ゲーム作成が可能となりました。
